# CODSOFT Tech Internship

This repository contains the tasks completed as part of the CODSOFT Tech Internship program. Each project is implemented using Python and follows the given guidelines.

##  Task 1 – To-Do List Application
Description:
A simple command-line based To-Do List application that allows users to add, view, and manage their daily tasks efficiently.

Key Features:
- Add new tasks
- View existing tasks
- Organize tasks in a structured list

Technology Used:
- Python

---

##  Task 2 – Calculator
Description:
A basic calculator application that performs fundamental arithmetic operations such as addition, subtraction, multiplication, and division.

Key Features:
- User input-based calculations
- Supports multiple arithmetic operations
- Displays accurate results instantly

Technology Used:
- Python

---

##  Task 3 – Password Generator
Description:
A password generator program that creates strong and random passwords based on user-defined length to improve security.

Key Features:
- Generates random secure passwords
- User-defined password length
- Uses Python’s random module

Technology Used:
- Python
